Evaluación 3a Unidad con METRO UI y CodeIgniter (PHP framework)
----

1. La carpeta debería llamarse metro_eval y estar ubicada directamente en htdocs (xampp),
   para ser accedida luego con ña ruta http://localhost/metro_eval/, esto debido a la
   configuración de las rutas del framework.

2. Use el modelo bd_eval de MySQL. La configuración de la base de datos deja acceder a root
   sin contraseña.

----

ARCHIVOS DE CONFIGURACIÓN DE CODEIGNITER:

- Base de datos:
application/config/database.php

- Ruta de acceso:
